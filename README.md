# Happy Birthday

친구에게 생일 축하 웹 페이지를 만들어주었다.<br></br>
`html`, `css`, `javascript`를 공부할 수 있었다.

## 작동방식
- 친구에게 문제를 내는 식으로 진행된다.
- 선택지는 두 가지로, 각각 친구에 대한 칭찬과 욕설로 구성된다.
- 칭찬을 선택하면 정답이다.
- 욕을 하면 정답이 아니다.
- 정답을 모두 맞추면 생일 축하 메시지와 사진이 뜬다.
- 한 문제라도 틀리면 다시 문제를 풀라는 메시지와 사진이 뜬다.

## 개발 의도
- 친구가 스스로에게 칭찬을 하면 좋을 것 같았다.
- 재미있는 장난감을 개발하고 싶었다.
